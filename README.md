Виды вопросов
---

+ Прямой ввод значения:
	+ Строка
	+ Текст
	+ Целое число
		+ Прямое указание
		+ В интервале
	+ Дробное число
		+ Прямое указание
		+ В интервале
	+ Дата
	+ Подтверждение (чекбокс)
+ Выбор варианта:
	+ Выбор одного из вариантов:
		+ Целое число
		+ Дробное число
		+ Строка
		+ Картинка
	+ Выбор нескольких вариантов
		+ Целое число
		+ Дробное число
		+ Строка
		+ Картинка
+ Интервал
	+ Из целых чисел
	+ Из дробных чисел
+ Роутер
	+ Целое число
	+ Дробное число
	+ Строка
	+ Картинка
